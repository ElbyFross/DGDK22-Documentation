Doloro.UIEngine2.Modules.ElementsCollection.AGUIElementsCollection Class Referenceabstract

Detailed Description

Base class for any UI element that handling bunch of another elements and should provide navigation among them.

Implements automatically feature of disabling elements out of the scope.

◆ eNoticedUnityHierarchyPositionChange

Action<GUIElement, Transform, int> Doloro.UIEngine2.GUIElement.eNoticedUnityHierarchyPositionChange
inherited

Occurs when system handler noticing that the place of GUI element in Unity-site hierarchy is changed.

Arguments

GUIElement - element itself. Transform - new parent. int - new child index at the parent child hierarchy.

Note
