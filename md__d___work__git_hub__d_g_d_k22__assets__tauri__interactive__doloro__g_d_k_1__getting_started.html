Letter to customer

Please take a note that the GDK contains a lot of modules. Each module is a separated part that could reference another one, but by itself has an isolated architecture.

Doloro GDK is a big framework. Before digging into depths of documentation we recommend exploring the common structure with brief overview by following link: modules_overview.

We also strongly recommend you use online documentation other than offline to get most relevant instruction. Offline documentation updates with delay.
