About

For certain tasks you may need to implement some custom pathfinder algorithms other than supplied with the module.
Doloro Graphs not interferes with such extension. You may simply implement fully integrating custom algorithms as you need.

Implementation

To create a new path algorithm you have to create a new class derived from Doloro.Graphs.Path class. To implement your own pathfinder algorithm just override following handlers along with your logic.

  • Doloro.Graphs.Path.Build
  • Doloro.Graphs.Path.BuildAsync
  • Doloro.Graphs.Path.BuildCoroutine

Following pattern is a best way to implement the pathfinder class.

using System.Collections;
using System.Collections.Generic;
using System.Threading.Tasks;
using Doloro.Graphs;
using UnityEngine;
public class CUSTOM_PATH : Path
{
public CUSTOM_PATH(GraphNode from, GraphNode to) : base(from, to)
{
// Your initialization here.
}
public override ICollection<GraphNode> Build()
{
var _t = Task.Run(BuildAsync);
_t.ConfigureAwait(false);
return _t.Result;
}
public override IEnumerator BuildCoroutine()
{
// Attempting to get cache using base path implementation.
var _cache = base.BuildCoroutine();
// Checking is the cached path has been found.
if (_cache != null)
{
// Path is found in cache and following the same type
// as algorithm itself.
if(_cache is CUSTOM_PATH cachedPathIsntance)
{
// Todo handling cache as instance of the same path type.
}
}
// Path have to be computed.
else
{
// Marking path state as `InProgress`
state = State.InProgress;
// Computing path ---------------------------------
//
// Between search stages (like nodes changes)
// don't forget to set `yield return null;` or more
// applicable alternative to share the CPU time with our tasks.
// ------------------------------------------------
....
// ------------------------------------------------
}
// Claiming path as failed in case if search loop did not complete.
if (state != State.Built)
state = State.Failed;
}
public override async Task<ICollection<GraphNode>> BuildAsync()
{
// Attempting to get cache using base path implementation.
var _cache = await base.BuildAsync();
if (_cache != null)
{
if(_cache is CUSTOM_PATH cachedPathIsntance)
{
// Todo handling cache as instance of the same path type.
}
return _cache;
}
// Turning on path computing state.
state = State.InProgress;
// Doing search --------------------------------
//
// Don't forget to switch treads between search stages
// by calling `await Task.Yield();`.
// ---------------------------------------------

// ---------------------------------------------
// Don't forget to conclude state as `Built` or `Failed`
// after search.
}
}
