About

A system that implements clustered 2 dimensional database with.

Features

  • Fast element access: O(1).
  • Economic RAM using.
  • Fast allocation (database for 400 000 000 element allocates in <1ms).
  • Endless data space (you could safety access any index of the table).
  • Unity-site table editor
  • XML 2003 exporting (friendly with Excel)

Structure

Table (Database)

Raw 2 dimensional data set.

Provides elements read\write access in O(1) for existing clusters. New clusters initialization requires O(EXIST_CLUSTERS_N) time.

Supports binary serialization.

Spreadsheet (Project)

Meta file that contains setting referenced to certain table that defines high-end behavior.

Main properties:

  • Columns\rows sizes
  • Columns\rows titles
  • User scope position
  • Styles
  • Reference to the source data set.

Solution (Complex workspace)

Meta file that bonds different spreadsheets to single project.

  • NOT IMPLEMENTED Implements API for bridging data between different tables.
  • Manages spreadsheets loading.

SpreadsheetEditor

Editor that displays UI for single Table + Spreadsheet pair.

SolutionInspector

Editor that manages data provides with the Solution instance.
