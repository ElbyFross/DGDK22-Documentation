Using

Resources configuration

The module has several template resources that settings up main entities participating graph rendering process.

Resource Purpose
ClusterGraphSchema GUI element for ClusterGraph elements. Using as main prefab during API calls. Implements GUI layout rules and bind world points to backstage graph's nodes.
ScalableLineGraph GUI element for ScalableLineGraph. Implements GUI layout rules and bind world points to backstage graph's nodes.
UnitGraph Part using in ScalableLineGraph resource that defines a single unit schema.

Package related resources located by following in-project path:
Assets \ Doloro-GDK \ Doloro-UI-2-Graphs \ Runtime \ DoloroUI2 Graphs \ Resources \ Templates

All the resources can be duplicated and reconfigured along with your GUI style for continue use. Make sure you are changed connected prefabs references and not refer on duplicate the old reference.


API

Graph to UI rendering goes in encapsulated way. All you need is:

  1. Create \ Configurate \ Select using ClusterGraphSchema resource.
  2. Build Graph declaring connection direction during connection using following pattern:
    Doloro.Graphs.Direction DIRECTION = ... // Connection direction.
    FROM_NODE.Connect(TO_NODE, DIRECTION);
  3. Send the data and resource to UniformGraphUI.Create API method.


Example

In following example, we will create and initialize data graph and then instantiate GUI from it.

using UnityEngine;
using Doloro.Graphs;
using Doloro.UIEngine2.Modules.Graphs
// Demo script that generates on-scene graph GUI.
//
// `guiSettingsPrefab`: Resource that defines graph GUI parameters.
public void CreateOnSceneGraph (ClusterGraphGUIElement guiSettingsPrefab)
{
if(guiSettingsPrefab)
// Creating a graph.
var data = new Graph();
// Creating graph nodes.
data.Add("c0", out var c0);
data.Add("p0", out var p0);
data.Add("p1", out var p1);
data.Add("c1", out var c1);
data.Add("c2", out var c2);
data.Add("h0", out var h0);
// Defining connections within the graph.
//
// Notice that we are declaring from which side
// connecting node connected to source node.
//
// Sample graph schema:
//
// p0---p1
// |
// c1-- |
// |--c0--h0
// c2--
//
c0.Connect(p0, Direction.Top);
c0.Connect(p1, Direction.Top);
p0.Connect(p1, Direction.Right);
c0.Connect(c1, Direction.Left);
c0.Connect(c2, Direction.Left);
c0.Connect(h0, Direction.Right);
// Calling GUI generation for them.
var GUILaout = UniformGraphUI.Create(
guiSettingsPrefab, // GUI prefab
data, // Data graph
c0.signature); // Node that will be selected as origin
}
