About

The items has implemented AGUIElementsCollectionDataConnector class supplied with the dui2 Elements Collection module. It allows to encapsulate bond between data set and its GUI representation.

Explore API:

  • Doloro.InvetorySystem.UI.DataBridge.ItemsGUICollection : ItemsCollectionDataConector
  • Doloro.InvetorySystem.UI.DataBridge.ItemsCollectionDataConector
  • Doloro.UIEngine2.Modules.ElementsCollection.AGUIElementsCollectionDataConnector{SourceType, GUIElementType}

Controls

From the box you has two options to use: ItemsCollectionDataConector and ItemsGUICollection.

  • ItemsGUICollection has solid layout requirements brings ItemSelectionEventRelay and GridCollection components with it. This option is preferred in common cases.
  • ItemsCollectionDataConector is more abstract class that not defines precise GUI controls to manage. Use in case you require specific layout made with non standard controls.
Warning
Components add not configure entire required layout controls from adding. Use configured prefabs as source or define the layout manually to use.
