Detailed Description

The base class for any integrable patchers. Will be called by the Patcher instance during validation pass for each node in a project.

State Doloro.AI.DAIDebugger.PatchingTool.APatchLogic.Patch ( PipelineMap  node,
PatchTicket  ticket 
)

Starts patching sequence.

Parameters
node
ticket
Returns
  • Sequence suspending with State.Failed in case if OnPrePatch(PipelineMap, PatchTicket) caused exception.
  • OnPostPatch(PipelineMap, PatchTicket) calling in any case.
  • OnPatch(PipelineMap, PatchTicket) executes only in case if Result equals to State.Succeed after the OnPrePatch(PipelineMap, PatchTicket) handler.
  • Result will be the State.PartialySuccesed in case if OnPostPatch(PipelineMap, PatchTicket) failed.
