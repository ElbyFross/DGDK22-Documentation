Window Auto Size

About

An util component that allows to compute the Window min and max sized along with content in auto mode.

Limits

An util considers that entire hierarchy including virtual one built with using of RuntimeWorkspaceLayout module and not using some custom layouts direved directly from the AClampedWorkspaceProvider.

How to use

Just add WindowAutoSizeClamp component on the game object with applied Window derived component. After that it'll automativaly recomute entire layout in first LateUpdate.

You can call manual update of layout with WindowAutoSizeClamp.Recopute() handler.
