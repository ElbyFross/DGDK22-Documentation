Plug-in: Scene Placement

About

A small plug-in that useful in case when you are using scene to prefabs configuration.
The tool allows to avoid endless manual adjustments with objects transform by binding the action to a single on-scene view button.

How to use

  1. Open Scene Placement tool at the Small Tools Hub window.
  2. Enable the tool with a toggle near to the catalog item related to it.
  3. Define origin object as transform source or static settings to apply.

No you may select object\objects on scene and apply new transform to it by pressing To place point button at the STH toolbar over the scene view.
