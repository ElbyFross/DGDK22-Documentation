Doloro.UIEngine2.AGUIElement Class Reference

Detailed Description

Base class of all UI elements int Doloro UI Engine.

GUI elements exist in common world space coordinates that starting at the left-bottom screen corner at the Unity. At the same time some GUI features of Unity using screen coordinates starting at the left-top corner of the screen. Moreover Unity could affect engine with the different anchor settings.

All this differences could cause bugs. Be careful. The engine always use Transform.position parameter during layout handle and considers start coordinates as left-bottom.
