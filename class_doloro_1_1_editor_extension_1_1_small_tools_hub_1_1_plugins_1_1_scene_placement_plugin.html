Doloro.EditorExtension.SmallToolsHub.Plugins.ScenePlacementPlugin Class Reference

Detailed Description

Provides extra control to placing object to certain world position.

Useful when you need to place huge bunch of objects to the same scene point. Reduces extra manual input of the coords.
