Doloro.AI.DAIDebugger.PatchingTool.AssemblyPatchingTool.AssemblyPatcher Class Reference

Detailed Description

Provides assembly validation and fixing of loosed\modified types referenced by projects. Allows to remake or drop defined property that can't be instantiated.

PatchTicket.sharedData map:

  • [0]: (PipelineMap.BinaryPropertyContainer) instance that can't be restored.
  • [1]: (TypesTransferCatalog.TypeSignature) of the type registered at the binary file.
  • [2]: (int) destination source id. 0: catalog, 1: manual
  • [3]: (TypesTransferCatalog.TypeSignature) manual destination type in case if selected manual option.
  • [4]: (int) index of data in source binary array.
