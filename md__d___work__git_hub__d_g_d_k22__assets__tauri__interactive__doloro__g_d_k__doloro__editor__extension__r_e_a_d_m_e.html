About

DEE is a module that extends Unity's editor features.
Using the module, you'll be able to create more flexible and beautiful tools spend lesser time than ever before.

More in this category:

Sub-module Description
Small Tools Hub Powerful tools hub that allows to integrate new
features faster.
Prevents editor from overwhelming interfaces when it's unnecessary.
Project Settings Toolkit Handful tool that allows fast configuration of editor
settings. Useful to supply some module with partial instructions,
avoiding override of a whole project settings files.

EditorGUILayout

DEE module implements new editor controls that will help you to make your editor tools better than ever before.
Explore API: Doloro.EditorExtension.DoloroEditorGUILayout

Work with new GUI controls goes the same way as with regular GUI elements supplied with the UnityEditor.EditorGUI and UnityEditor.EditorGUILayout.
