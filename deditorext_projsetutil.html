Project settings utils

The collection of plugin for Unity editor configuration.

Как работает:

Tools > Project Setting

> New - Создает в указаном месте asset. В котором можно задавать настройки для проекта. > Load - Находит все данные asset, после выбираете какие хотите загрузить и они добавляются в настройки.

Script:

PrjSt

Class : ScriptableObject, для хранения настроек проекта. Обязан рееализовывать определенный interface.

CreatePrjSt

Класс для создания asset из PrjSt.

ProjectSetting

Класс который содержит методы для нахождения файлов настроек и метод для добления этих настроек.

LayersSetting

Static class который реализует методы для добавления Layer в Tags & Layers

TagsSetting

Static class который реализует методы для добавления Tag в Tags & Layers

InputSetting

Static class который реализует методы для работы с Unity Input

InputAxis

Struct которая может хранит информацию Axis из Input

Editor:

ProjectSettingsNew

Окно в котором можно создать новый файл (asset) с насстройкками

ProjectSettingsWindow

Нходит все файлы настроек и дает возможность выбора тех что необходимо добавить.
