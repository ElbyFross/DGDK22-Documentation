About

The items has implemented AGUIElementsCollectionDataConnector class supplied with the dui2 Elements Collection module. It allows to encapsulate bond between data set and its GUI representation.

Explore API:

Controls

Warning
Components add via Add component menu not configures entire required layout controls from adding. Use configured prefabs as source or define the layout manually to use.

The package has several main AGUIElementsCollectionDataConnector variants:

Component Description
StoragesGUICollection A solid layout requirements brings
StorageSelectionEventRelay and GridCollection components with it.
This option is preferred in common cases.
StoragesCollectionDataConector A more abstract class that not defines precise GUI controls to manage.
Use in case you require specific layout made with non standard controls.

The package also has implementations related to sub-modules:

Component Sub-Module Description
TransferAgentCollectionDataConector Shops Configures part of ShopsInspector layout.
StoragesCollectionDataConector by itself.
