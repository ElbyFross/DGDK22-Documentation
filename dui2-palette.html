How to setup

Palette creation

Press LMB on Project panel, next select Create\Color Palette (DoloroUI2) line. Place created asset ibn subdirectory Resources\palettes to make it visible for PalettHandler service.

Using the asset you can define the collors to certain palette's positions.

Element connection

For connecting the element to the module you need to put the PaletteColorPicker component on the game object with the Image or Renderer component.

In case of Renderer attachment the picker will apply a binded palette color to the Renderer.color property.
