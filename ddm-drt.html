About

The system that handles loadable resource and encapsulate this management into simple use API.

Features

  • Loading resources to memory (including bundles content)
  • Search amoung resources with detailed queries
  • Search result hashing to prevent unecessary extra scans.
  • Auto memory management (not implemented)
  • Patching API
  • Assets merge API
